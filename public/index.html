URL Shortener Microservice

Example usage:

To create a new shortened URL: https://immense-everglades-70339.herokuapp.com/new/http://www.google.com
To visit a created URL: http://immense-everglades-70339.herokuapp.com/1234

Example creation output

{ long_url: "http://www.fark.com", short_url: "https://immense-everglades-70339.herokuapp.com/4435" }
